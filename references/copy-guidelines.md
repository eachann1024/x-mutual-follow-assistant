# Copy Guidelines (Simplified Chinese)

## Goal
Generate fresh copy every run while preserving this meaning:
- 蓝V用户在帖子下回复任意内容，并关注我后，我会按计划定时回关。

## Structure Options

### Template A
- 开场一句（吸引注意）
- 条件说明（蓝V + 回复 + 关注）
- 承诺说明（定时回关）
- CTA（现在就评论）

### Template B
- 标题行（短句+emoji）
- 规则 1 行
- 规则 2 行
- 收尾 CTA

### Template C
- 问句开场
- 条件与承诺并列
- 轻量提醒（非秒回，定时处理）

## Emoji Pool
Use 2-5 emojis each time; vary combinations:
- 🚀 🤝 ✅ 🔔 ⏰ 🌟 💙 📌 🎯

## Rotation Rules
- Do not repeat the same first sentence in consecutive runs.
- Alternate between short and medium length copy.
- Rotate CTA verbs: 回复 / 留言 / 评论 / 来一句。
- Rotate follow-back phrasing: 定时回关 / 分批回关 / 按计划回关。

## Example Variants

1. 🤝 蓝V互关计划开启！
在这条帖子下回复任意内容并关注我，系统会按计划定时回关你。⏰
来留言，我会分批处理～🚀

2. ✅ 蓝V朋友看这里：
只要你在本帖评论任意内容并先关注我，我会在固定时段回关。
不是秒回，但一定按节奏处理。🔔

3. 🌟 互相关注小约定
如果你是蓝V，评论区留个言并关注我，我会定时回关你。🤝
现在就来一条评论吧～🎯
